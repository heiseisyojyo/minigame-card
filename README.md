# 牌店买取狂人

一款像素风的卡牌店买取模拟小游戏。你需要在每日目标内完成卡片判定、查价与定价，控制失误和损失，提升卡店可信度。打开页面即可游玩，无需构建流程。

## 功能亮点

- **多日流程教学**：从 X 系列入门，到识别假卡、处理无法查价的卡片，再到使用 JHS APP 一键识图。
- **买取流程还原**：翻面查看 → 判定真假/品相 → 查价定价 → 进入下一张。
- **统计与反馈**：实时显示进度、累计失误、收益/损失、信誉度、用时等数据。
- **像素风界面**：主面板、工具栏与叠层面板统一像素风格。

## 快速开始

### 本地运行

1. 克隆或下载仓库。
2. 用浏览器直接打开 `index.html`。

> 建议使用现代浏览器（Chrome/Edge/Firefox）以获得完整动画与样式效果。

## 操作说明

- **开始游戏**：标题页点击“开始游戏”。
- **翻面查看**：点击“翻面查看”或直接点击卡片。
- **判定品相**：选择 A/B/C 或“假卡”。
- **查价定价**：根据系列选择 X 或 Y 查价网站，选定品相后确认价格。
- **JHS APP**：第 7 天解锁，点击“拍照识图”快速获取信息。
